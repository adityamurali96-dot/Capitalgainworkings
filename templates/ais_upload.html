AIS Reconciliation

Upload your capital-gains / broker file(s) — one or more, up to {{max_files}} — and the single AIS file. Columns are auto-detected; you then get a quick screen to confirm the AIS file's columns (the broker/CG files are read automatically). Sale values are totalled per security across every broker file and matched against AIS (ISIN first, then name).

Reconciliation compares sale value per security on each side. Several broker files are pooled per security (by ISIN, then name) first, so an account split across files still nets against the one AIS line. A security with an ISIN on one side and only a name on the other is matched by normalised name. Quarters/lots are summed; the preparer judges every delta.
